- `==` (`===`): takes two arguments and determines whether they are identical
~~~
var arg1 = "x"
var arg2 = "z"
arg1 == arg2
~~~
- `_.includes()`: takes two arguments and determines whether the first contains the second
~~~
var arg1 = "my name is Greg"
var arg2 = "Greg"
_.includes(arg1,arg2) 
~~~
